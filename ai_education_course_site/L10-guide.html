Phase 03

Phase 3：配置 Agent 提示词

总提示词和功能提示词分开，方便按钮调用。

【总提示词】
你是一个面向国内中学生的个人学习仪表盘 Agent。你的任务是把学生输入转化为结构化学习建议、学习记录和复盘结果。

【分析问题】
请输出：问题类型、卡点判断、一个提示、下一步任务、是否保存记录。

【生成今日计划】
请根据目标、可用时间和薄弱点输出 3-5 个任务，每个任务包含时间和完成标准。

【本周复盘】
请根据学习记录输出高频卡点、进步点、下周建议和最小行动。
Phase 04

Phase 4：工作流节点与连线

展示节点和连线方式，课堂实现时可使用模板。

Start：按钮输入接收按钮类型、学科、问题和历史记录
↓
Router：任务判断problem_analysis / daily_plan / weekly_review
↓
Agent Node：生成结果按功能提示词输出结构化内容
↓
Save Node：保存记录需要保存时写入学习记录
↓
Output：返回网页结果卡、任务清单或复盘摘要
Phase 05

Phase 5：参数设置

最终工具要稳定，参数不要太发散。

参数推荐原因
生成随机性0.3-0.5保证结构化输出稳定
携带上下文轮数5-8支持连续学习场景
最大回复长度1500-2200足够生成计划和复盘
输出格式固定字段方便网页展示和保存
