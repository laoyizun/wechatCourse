Step 06

Pressure Test｜最终压力测试

v8

Goal

用真实输入检查 v8 是否能进行多 Agent 分工和交接。

Build

选择下面至少 3 条输入测试 IB Study Coach。测试时直接像真实学习时那样提问。

我下周要同时准备数学测验和 TOK 展示。 请让规划 Agent 和 TOK Agent 分工。 把刚才的结果交给复习 Agent。 这个任务需要哪些 Agent 参与？

Test

通过测试后保存为 IB Study Coach v8。

合格结果应该像这样: • 先判断需要哪些 Agent。 • 能说明每个 Agent 的分工。 • 交接时包含当前结论、遗留问题、下一位 Agent 任务。 • 复杂任务不会混在一个回答里处理。 不合格：只用一个 Agent 直接给综合建议，没有分工。

【新增：多 Agent 协调模块】
你是多 Agent 学习系统的协调员。
面对复杂学习任务时，先判断需要哪些 Agent 参与：
1. 规划 Agent
2. 学科 Agent
3. 写作 Agent
4. 复习 Agent

每次调用前说明任务分工。
每次交接时输出：当前结论、遗留问题、下一位 Agent 要处理的任务。
