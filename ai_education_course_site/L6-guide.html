Phase 01

认识基础节点

先只使用最小 Demo 需要的节点，不急着把所有节点都加进来。

节点本节用途
输入接收学生问题和学科
大模型判断任务类型、生成分支回答
选择器根据 task_type 走不同分支
变量赋值保存某个中间字段
变量聚合把不同分支结果统一成 final_answer
文本处理整理最终输出格式
输出返回 final_answer
Phase 02

配置输入节点

输入字段要少而清楚，便于后续节点引用。

字段类型示例
user_question文本这道函数题我不会
subject文本数学
测试输入：函数单调性怎么复习？
Phase 03

添加大模型任务判断节点

第一个大模型节点只做任务判断，不负责正式回答。

你需要判断学生输入的学习任务类型。

学生输入：
{{user_question}}

请判断它属于：讲题、学习计划、知识点复习、其他。
Phase 04

强化 JSON 输出格式

如果大模型输出不能被解析，后面的选择器和变量节点就会失败。

你需要判断学生输入的学习任务类型。

学生输入：
{{user_question}}

请严格输出 JSON，不要输出任何解释、Markdown、代码块或多余文字。

输出格式必须是：
{
  "task_type": "讲题",
  "reason": "判断原因",
  "need_resource": true
}

字段说明：
- task_type 只能填写：讲题、学习计划、知识点复习、其他。
- reason 用一句话说明判断依据。
- need_resource 表示是否需要调用资料。

要求：
1. 只能输出一个合法 JSON 对象。
2. 所有 key 必须使用英文双引号。
3. 字符串值必须使用英文双引号。
4. 布尔值只能使用 true 或 false。
5. 不要使用 Markdown 代码块。
6. 不要在 JSON 前后添加任何说明文字。
7. 如果无法判断，task_type 填写 "其他"。
Phase 05

测试 JSON 输出

先确认 JSON 稳定，再继续接选择器。

输入期望 task_typeneed_resource
这道函数题我不会。讲题true
帮我安排今晚 30 分钟数学复习。学习计划false
函数单调性怎么复习？知识点复习true
我不知道怎么学。其他false
Phase 06

添加选择器节点

选择器读取 task_type，然后走不同分支。

条件分支目标
task_type = 讲题讲题大模型节点
task_type = 学习计划计划大模型节点
task_type = 知识点复习复习大模型节点
其他补充提示节点
Phase 07

添加分支大模型节点

每个分支的大模型只处理自己的任务，不要写成一个万能 Prompt。

分支Prompt 要点
讲题判断卡点，给分层提示，不直接给完整答案
学习计划根据时间生成任务、时长、完成标准
知识点复习给复习步骤、例题方向、检测方式
其他请学生补充学科、题目或目标
Phase 08

变量聚合与文本处理

不同分支会产生不同回答，用变量聚合统一成 final_answer，再用文本处理整理格式。

讲题 answer
↓
计划 answer
↓
复习 answer
↓
变量聚合 final_answer
Phase 09

配置输出节点

输出节点只返回最终字段，不要把中间调试字段全部暴露给学生。

输出字段：
final_answer

建议输出结构：
任务判断：
关键帮助：
下一步：
Phase 10

JSON 失败修复清单

解析失败时，优先修大模型节点提示词。

检查项修复方式
是否输出 Markdown 代码块要求不要使用代码块
JSON 前后是否有解释要求只能输出一个 JSON 对象
key 是否用了英文双引号明确所有 key 必须双引号
字段名是否一致固定 task_type、reason、need_resource
布尔值是否写成字符串true/false 不加引号
是否输出多个 JSON要求只输出一个对象
Phase 11

完整 Demo 测试

测试时记录 task_type、选择器分支、final_answer 三个结果。

测试输入应走分支合格表现
这道函数题我不会。讲题给分层提示，不直接给完整答案
帮我安排今晚 30 分钟数学复习。学习计划输出任务和时间安排
函数单调性怎么复习？知识点复习输出复习步骤和检测方式
我不知道怎么学。其他要求补充目标或学科
Phase 12

循环节点轻量拓展

本节只认识循环的用途：对多个对象重复执行同一种处理。

拓展示例：
输入 weak_points = ["函数定义域", "函数单调性", "二次函数图像"]
循环：依次为每个知识点生成一个 10 分钟复习任务
变量聚合：合并成今日复习清单
