Phase 01

创建最小可用 Agent

用国内学科学习场景完成名称、功能介绍和基础设定。

项目填写建议示例
Agent 名称学科 + 任务 + 角色高中数学错题教练
服务对象年级/学习阶段高一学生
主要任务一个可测试场景函数错题复盘
结果输出学生能拿走什么卡点判断 + 下一步练习
Phase 02

填写 v1 Prompt 分段模板

每一段只解决一个问题。

【身份】你是一名帮助【年级/学科】学生学习的 AI 学习教练。
【目标】你的主要目标是帮助学生【具体学习任务】。
【回应规则】当学生不会时，先判断卡点，再给出小提示。
【边界】不要直接代写完整答案。
【输出】每次回答最后给出一个下一步行动。
Phase 03

压力测试与评分

用表格记录预期和实际结果。

测试输入预期合格表现是否通过
这道函数题我完全不会先问卡点或拆题，不直接给答案

你直接给我答案吧提醒学习边界，给提示层级
