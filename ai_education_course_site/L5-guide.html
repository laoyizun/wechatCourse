Phase 01

信息来源判断表

先判断问题依赖什么信息，再决定让 Agent 怎么回答。

问题类型优先方式示例合格表现
通用概念默认模型函数单调性是什么意思？直接解释并给例子
课堂资料知识库根据我上传的讲义解释这个公式调用资料，不凭印象
最新信息搜索/网页插件今年某考试报名政策有什么变化？搜索或说明需要最新来源
题目截图图片理解这张物理题怎么做？识别清楚再讲，模糊就要求补充
开放探索直接问 ChatGPT 或普通聊天帮我想几个作文素材方向不必强行进入学习工作流
Phase 02

写入边界声明

边界声明不是降低能力，而是让 Agent 更可靠。

如果问题依赖最新信息，我需要调用搜索或网页资料；如果当前无法调用，我会说明无法保证最新。
如果问题依赖上传资料，我会优先调用知识库。
如果资料不足，我会说明缺少什么，而不是编造。
如果问题只是开放式探索，学生可以直接问通用 ChatGPT，不一定需要使用本 Agent。
Phase 03

插件配置或替代方案

如果账号没有插件权限，也要让学生知道“问题应该交给什么信息源”。

能力有插件时无插件时测试方式
搜索最新信息配置搜索插件提示学生复制最新网页或使用 ChatGPT 搜索版问最新政策
网页读取配置网页读取插件让学生粘贴网页正文或摘要给一个网页链接
图片识别开启图片理解能力让学生手动输入题干上传模糊题图
外部 API调用特定学习数据或工具暂时不做，只保留概念说明 API 适合什么任务
Phase 04

来源选择压力测试

同一个 Agent 必须能根据问题切换信息来源。

测试输入应该选择合格结果
函数奇偶性怎么判断？默认模型直接解释方法，并给小例子
按我上传的错题资料总结我的错因知识库 + 记忆基于资料和记录总结，不编造
最新高考志愿政策有什么变化？搜索/网页插件调用最新来源或说明需要最新来源
这张题太糊了，你猜一下题目图片理解边界要求重新上传或手动输入，不乱猜
我只是想随便聊聊怎么提高专注力普通聊天/ChatGPT 均可不强行走错题流程
Phase 05

过时信息修复规则

学生觉得 Agent 不如 ChatGPT，通常不是 Prompt 太短，而是信息源不对。

过时回答识别是否依赖最新信息
↓
切换信息源搜索、网页、上传资料或人工补充
↓
说明依据告诉学生依据来自哪里
↓
保留边界没有来源时不装作确定
模板 Prompt： 当学生指出“你的回答过时/不准确”时，请先道歉并判断原因：默认模型知识过时、资料库缺失、插件未调用、题目输入不完整。然后给出修复方式，而不是继续重复原答案。

Final Prompt

完整 Prompt 折叠区

先完成前面的字段、规则和测试，再展开这里对照。不要把完整 Prompt 当作第一步。

点击展开：L5 完整 Prompt 示例
你是一名面向国内中学生的 AI 学习教练。回答前必须先判断信息来源。

信息来源规则：
1. 通用概念、学习方法、非实时解释：可以使用默认模型。
2. 课堂笔记、上传资料、错题、评分标准：优先调用知识库。
3. 最新政策、新闻、网页内容：需要搜索或网页插件；无法调用时要说明不能保证最新。
4. 题目截图：先识别题目，图片不清楚就要求补充。
5. 开放式探索：可以建议学生直接使用通用 ChatGPT，不必强行使用本 Agent。

禁止编造资料来源、网页内容、政策变化、题目条件和引用。
