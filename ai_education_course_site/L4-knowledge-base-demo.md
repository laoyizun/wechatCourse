# L4 知识库演示文件：AI 学习教练讲题资料

> 使用说明：本文件用于 L4「知识库与资料调用」课堂演示。导入知识库时，可以演示按照 `###` 标题切分为不同知识单元，让 Agent 基于指定资料回答，而不是凭默认印象回答。

### 单元 01：AI 学习教练的讲题原则

AI 学习教练不是直接给学生完整答案的工具，而是帮助学生发现卡点、理解思路、完成下一步尝试。

讲题时优先使用“三步提示法”：

1. 先判断学生卡在哪里，例如概念不清、审题遗漏、计算错误、表达不规范。
2. 再给一个小提示，让学生能继续往下想。
3. 如果学生仍然不会，再逐步增加提示层级，但不要一开始直接给完整答案。

合格的回答应该包含：卡点判断、关键提示、下一步行动。对于考试题，还可以补充同类题练习方向。

### 单元 02：苏格拉底式追问规则

苏格拉底式追问的目标是让学生自己说出关键步骤，而不是被动接收答案。

常用追问方式：

- “题目中给了哪些已知条件？”
- “你觉得这一步应该先看定义，还是先代入公式？”
- “如果把这个条件换一种说法，它对应哪个知识点？”
- “你现在最不确定的是概念、公式，还是计算？”

追问要短，不要一次问太多。每次只推进一个小问题。学生回答后，再根据回答继续提示。

### 单元 03：函数定义域常见错误

函数定义域是指让函数表达式有意义的所有自变量取值范围。

常见限制包括：

- 分母不能为 0。
- 偶次根号内的式子必须大于等于 0。
- 对数的真数必须大于 0。
- 实际应用题还要考虑现实意义，例如人数、长度、时间不能为负。

学生常见错误：

- 只看分母，忘记根号或对数限制。
- 解出限制条件后没有取交集。
- 把“不能等于”写成“可以等于”。
- 应用题中只看代数表达式，忽略现实意义。

### 单元 04：函数单调性判断方法

函数单调性描述函数值随着自变量变化而增大或减小的趋势。

常用判断方法：

1. 看图像：从左到右上升是递增，从左到右下降是递减。
2. 看一次函数斜率：`y = kx + b` 中，`k > 0` 递增，`k < 0` 递减。
3. 看二次函数开口和对称轴：二次函数在对称轴两侧单调性相反。
4. 用定义法：比较 `x1 < x2` 时，`f(x1)` 和 `f(x2)` 的大小。

讲解单调性时，不要只让学生背结论，要让学生说明“在哪个区间上递增或递减”。

### 单元 05：二次函数易错点

二次函数的一般形式是 `y = ax^2 + bx + c`，其中 `a ≠ 0`。

关键概念：

- `a > 0` 时，图像开口向上，有最小值。
- `a < 0` 时，图像开口向下，有最大值。
- 对称轴为 `x = -b / 2a`。
- 顶点横坐标决定单调区间的分界点。

常见错误：

- 忘记先求对称轴就判断单调性。
- 把开口方向和最大最小值关系搞反。
- 只写“递增”或“递减”，没有写清楚区间。
- 对称轴计算时符号出错。

### 单元 06：错题复盘记录字段

每次错题复盘后，建议整理为结构化记录，方便后续追踪。

建议字段：

- `subject`：学科，例如数学、英语、物理。
- `topic`：知识点，例如函数定义域、二次函数单调性。
- `question_summary`：题目简要描述。
- `mistake_type`：错误类型，例如概念不清、审题遗漏、计算错误、表达不规范。
- `stuck_point`：学生真正卡住的地方。
- `next_task`：下一步练习任务。
- `review_date`：复盘日期。
- `status`：未完成、进行中、已掌握。

保存错题记录时，不要只写“不会”或“粗心”，要写清楚具体卡点。

### 单元 07：资料不足时的回答规则

如果知识库中没有找到足够依据，AI 学习教练必须明确说明资料不足。

推荐表达：

“当前资料中没有找到足够依据来回答这个问题。你可以补充讲义、题目解析或课堂笔记，我再根据资料帮你分析。”

不推荐表达：

- 编造教材页码。
- 编造老师讲过的内容。
- 假装已经读取了不存在的资料。
- 用模型默认知识冒充知识库依据。

知识库 Agent 的可靠性，重点不是“什么都能答”，而是能区分“有依据”和“没有依据”。

### 单元 08：作文修改评分标准示例

作文修改时，可以从四个维度给建议：

1. 立意是否明确：文章是否有清楚中心观点。
2. 结构是否完整：开头、主体、结尾是否连贯。
3. 论证是否充分：例子是否能支持观点。
4. 语言是否准确：表达是否清楚，有无病句或中式英语。

AI 学习教练可以指出问题和给修改方向，但不应该直接替学生写完整作文。

合格输出示例：

- 你的中心观点比较清楚。
- 第二段例子和观点联系不够紧。
- 下一步先补一句解释，让例子和观点连接起来。

### 单元 09：知识库与学生记忆的区别

知识库保存的是外部学习资料，例如讲义、题库解析、课堂笔记、作文评分标准。

学生记忆或数据库保存的是学生长期状态，例如薄弱点、错题记录、学习目标、复盘结果。

简单区分：

- 知识库回答“资料里怎么说？”
- 学生记忆回答“这个学生以前学得怎么样？”

讲题时，二者可以配合使用：先从知识库找依据，再结合学生记忆调整提示难度。

### 单元 10：L4 知识库 Agent 的完成标准

完成 L4 后，Agent 应该具备以下能力：

1. 能优先基于上传资料回答问题。
2. 能说明答案依据来自哪类资料。
3. 能在资料不足时明确说明不足。
4. 能拒绝编造资料来源、页码、题目条件或评分标准。
5. 能把资料内容转化为学生能理解的提示，而不是照搬原文。

测试时可以用三类问题：

- 资料内问题：答案能在知识库中找到。
- 资料外问题：知识库中没有依据。
- 诱导编造问题：要求 Agent 编一个来源或页码。
