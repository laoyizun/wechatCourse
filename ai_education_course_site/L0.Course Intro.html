L0 · Course Intro

AI学习系统设计课
课程介绍

这门课面向国内初高中与高考体系学科学习，目标是把 AI 从聊天工具升级为自己的学习系统。

国内学科Coze Agent学习工具仪表盘项目
Why

为什么要学这个

学生已经会问 AI，但真正有用的是把 AI 变成稳定、可复用、可记录的学习工具。

不是

Prompt 小技巧合集

而是

持续升级一个学习 Agent

最终

做出自己的学习仪表盘工具

Main Line

课程主线：学习 Agent 版本升级

每节课让同一个学习 Agent 增加一种真实能力。

v1-v2Prompt、行为、参数
↓
v3-v5记忆、资料、插件和边界
↓
v6-v8单 Agent / 多 Agent 工作流与整合
↓
Project个人学习仪表盘
Subjects

样例以国内学科为主

所有课堂样例会围绕语文、数学、英语、物理、化学、生物、历史、地理等学习任务。

01数学：函数错题复盘
02英语：阅读长难句分析
03语文：作文结构修改
04物理：受力分析流程
05化学：方程式配平
06历史：材料题答题结构
Course Map

Outcome

最终你会得到什么

不是一个“看起来会聊天”的机器人，而是一个能服务自己学习流程的工具。

可配置

Prompt、参数、记忆、知识库。

可调用

资料、插件、工作流。

可记录

学习问题、薄弱点、任务。

可展示

仪表盘页面和使用流程。
