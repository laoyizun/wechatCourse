Chapter 07

Multi-Agent：多智能体协同

理解 AI 团队分工，不是多个 AI 聊天。

大尊老师
核心内容

Multi-Agent 不是多个 AI 聊天

  • 多个 AI 一起聊天不等于协同。
  • 真正的 Multi-Agent 是任务分工、上下文共享、结果汇总。
  • 要有 Leader，要有分工，要有交付标准。

知识点

常见 Agent 分工

Leader Agent

拆任务、分配任务、汇总结果

分析 Agent

理解数据、用户和问题

内容 Agent

生成文案、报告、方案

Reflection Agent

Memory Agent

流程

多智能体任务流程

1Leader 接收任务
2拆分子任务
3分配给子 Agent
4共享上下文
5汇总结果
6Reflection 复盘

核心内容

教育系统中的多 Agent

  • AI导师：负责讲解和答疑。
  • 练习Agent：负责出题和反馈。
  • 报告Agent：负责学习总结。
  • 记忆Agent：负责学生画像。
  • 反思Agent：负责教学策略优化。

总结

Multi-Agent：多智能体协同 的核心结论

Multi-Agent 的价值，不是数量多，而是分工清晰、上下文统一、结果可汇总。
