核心认知

Prompt 解决一次对话，Agent 解决长期工作

  • 学 Agent，不是为了做一个更会聊天的机器人。
  • 学 Agent，是为了把稳定的工作流程沉淀下来。
  • Agent 的核心价值是：岗位化、流程化、可复用。
为什么需要 Agent

普通 AI 聊天的问题

每次都要重新教 AI

  • 我们是谁
  • 做什么产品
  • 用户是谁
  • 风格是什么
  • 输出格式是什么

概念定义

Agent 到底是什么？

  • 一句话理解：Agent = 训练好的 AI 员工。
  • 它不是临时回答问题，而是按照设定好的岗位、流程和知识，持续完成某类任务。
  • 普通聊天 = 临时工；Agent = 培训好的正式员工。
Prompt vs Agent

Prompt 和 Agent 的区别

Prompt

  • 一次性需求
  • 适合临时任务
  • 结果依赖提问水平
  • 每次都要补充背景
  • 个人使用为主

Agent

  • 长期使用
  • 适合固定岗位
  • 结果更稳定
  • 可接入知识库
  • 适合团队复用
组成结构

一个 Agent 由什么组成？

1 大脑

系统提示词：决定怎么思考。

2 记忆

知识库：决定懂什么。

3 流程

工作流：决定先后顺序。

4 工具

能力扩展：能做哪些事。

5 结果

输出规范：结果长什么样。

工作流程

Agent 是怎么工作的？

1用户输入
2识别任务
3读取知识
4按流程处理
5规范输出

重点不是“能不能回答”，而是“能不能按照固定流程稳定交付”。

好坏对比

好 Agent 和坏 Agent 的区别

坏 Agent

  • 什么都能聊
  • 岗位不清晰
  • 输出不稳定
  • 风格经常变
  • 像一个普通聊天机器人

好 Agent

  • 只专注一个岗位
  • 流程清楚
  • 输出稳定
  • 有专业边界
  • 像一个可靠员工

海报设计提示词助手

用户输入

  • 活动主题

  • 风格要求
  • 配色方向
  • 图片比例
  • 投放平台

Agent 输出

  • AI绘图提示词
  • 画面布局建议
  • 主标题/副标题
  • 配色建议
  • 避坑提醒

核心定位：不是帮你画图，而是帮你把视觉需求讲清楚。

实践项目 2

招生咨询助手

客户问题

  • 零基础能学吗？
  • 考试难不难？
  • 可以入户吗？
  • 怎么收费？
  • 多久能学完？
  • 不过怎么办？

Agent 输出

  • 自然回复话术
  • 用户痛点回应
  • 课程推荐理由
  • 下一步引导
  • 避免过度承诺

核心定位：不是机械回答，而是理解咨询背后的真实需求。

学生练习

课堂练习：先填 Agent 设计表

  • Agent 名称：海报设计提示词助手 / 招生咨询助手
  • 服务对象：设计师、运营、招生销售、客服
  • 解决问题：把重复、高频、标准化的任务交给 AI 先完成初稿。
  • 工作流程：先识别需求 → 再分析场景 → 再调用知识 → 最后按格式输出。
  • 输出格式：必须固定，避免每次结果样式不同。
避坑提醒

做 Agent 最容易踩的 6 个坑

1 太贪心

想让一个 Agent 什么都做，最后什么都不稳定。

2 没岗位

不知道它到底是运营、设计还是销售。

3 没流程

只写身份，不写做事步骤。

4 没知识库

不懂公司产品，只能泛泛回答。

5 没格式

每次输出样子都不一样，团队不好用。

6 不测试

上线前没有用真实问题反复测试。

迭代思维

好 Agent 是训练出来的

1第1版：能回答
2第2版：有流程
3第3版：有知识库
4第4版：输出稳定
5第5版：团队复用

不要追求一次做完，而是用真实工作不断修正它。

总结

Agent 章节的核心结论

Agent 不是聊天机器人，而是可复用的 AI 岗位工具。先设计岗位，再写提示词；先明确流程，再接知识库；先稳定输出，再考虑复杂能力。
