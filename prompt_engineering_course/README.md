# AI工作流培训网页演示文稿

## 使用方法

1. 解压 zip 文件。
2. 打开 `index.html` 预览课程入口页。
3. 点击章节进入演示文稿。
4. 演示页支持键盘 ← → 翻页。

## 当前章节

- `prompt-engineering.html`：提示词工程 BROKE
- `agent.html`：Agent 智能体应用

## 部署到 GitHub Pages

将整个文件夹内容上传到 GitHub 仓库，开启 GitHub Pages 即可。

## 备注

- 大图页面已设置为页面宽度约 90%。
- 封面页已预留“大尊老师”署名，位置在翻页提示上方，避免重叠。
