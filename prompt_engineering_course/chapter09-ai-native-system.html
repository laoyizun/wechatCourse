从接入 AI 到 AI Native System

  • 接入 AI：系统里加一个聊天框。
  • AI Native：AI 成为业务流程核心。
  • 系统设计重点变成 Agent + Workflow + Memory + 数据闭环。

AI教育系统案例

  • 学生画像：记录基础、目标、偏好。
  • 学习路径：根据阶段动态调整。
  • 学习报告：阶段性生成反馈。
  • 记忆卡片：把知识转为复习内容。
  • 长期成长：每次学习都进入数据闭环。
