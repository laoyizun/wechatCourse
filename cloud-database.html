☁️ 云数据库

💡 什么是云数据库？
云数据库是将数据存储在远程服务器上，通过网络进行数据的增删改查。多个用户可以同时访问和操作同一份数据，实现数据的实时同步和共享。数据永远不会因设备更换而丢失。

⚖️ 优缺点分析
✅ 优点
  • 数据永不丢失，服务器备份
  • 多设备实时同步
  • 存储容量大
  • 支持多人协作
❌ 缺点
  • 必须有网络连接
  • 需要后端服务支持
  • 响应速度依赖网络
  • 可能产生云服务费用

💻 代码示例（云开发）
// 微信云开发数据库操作
// 初始化云数据库
const db = wx.cloud.database();

// 添加投票数据
db.collection('votes').add({
  data: {
    title: '最喜欢的运动',
    options: ['篮球', '足球', '游泳'],
    votes: { basketball: 12, football: 8, swim: 5 },
    createTime: db.serverDate()
  }
});

// 查询投票数据
db.collection('votes').get().then(res => {
  console.log(res.data);
});

// 更新投票（增加票数）
db.collection('votes').doc('voteId').update({
  data: {
    'votes.basketball': db.command.inc(1)
  }
});
