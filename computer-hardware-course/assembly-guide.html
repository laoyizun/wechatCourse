12 Assembly Steps

按照顺序完成装机

点击每一步展开操作。页面中的照片位置会在补充对应图片后自动显示。

01准备机箱与工具Prepare the case and tools›
  1. 准备十字螺丝刀和小盒，用来收纳螺丝。
  2. 平放机箱，确认电源线已经拔下。
  3. 触摸接地金属物释放静电，拿配件时只拿边缘。
  4. 核对主板铜柱位置，避免多余铜柱接触主板背面。
禁止：不要打开电源外壳，不要在通电状态下插拔内部配件。
02安装 CPUInstall the CPU · 教师监督›
  1. 打开 LGA 插槽压杆和金属压盖。
  2. 拿住 CPU 两侧边缘，寻找 CPU 和插槽上的三角标记。
  3. 对齐缺口和三角标记，把 CPU 平放进插槽。
  4. 确认 CPU 完全平整后，放下压盖并锁紧压杆。
停止信号：CPU 不能平放或需要用力压入时，方向可能错误。不要触碰插槽针脚。
03安装散热器Install the CPU cooler›
  1. 清理旧硅脂，并在 CPU 中央放置适量新硅脂。
  2. 让四个固定脚对准主板孔位。
  3. 按照对角顺序均匀固定散热器。
  4. 轻轻检查散热器没有松动。
  5. 把风扇线接入主板 CPU_FAN 接口。
CPU_FAN 没有连接时，电脑可能报警、过热或自动关机。
04安装 DDR3 内存Install RAM›
  1. 打开内存槽两端卡扣。
  2. 对齐内存金手指缺口与插槽凸点。
  3. 双手在内存两端垂直、均匀按下。
  4. 确认卡扣自动锁住，内存两端高度一致。
不要触摸金手指。缺口没有对齐时，不要按压。
05安装主板Install the motherboard›
  1. 安装主板 I/O 挡板，如机箱和主板需要。
  2. 确认铜柱与主板螺丝孔位置一致。
  3. 把后部接口对准挡板，再让主板落在铜柱上。
  4. 先轻轻定位所有螺丝，再逐步拧紧。
螺丝固定即可，不要过度拧紧。多余铜柱可能造成短路。
06安装电源Install the power supply›
  1. 根据机箱通风孔确定电源风扇方向。
  2. 把电源推入电源仓，对齐机箱后部螺丝孔。
  3. 用四颗螺丝固定。
  4. 暂时把线材放到机箱侧面，避免遮挡主板。
任何情况下都不要拆开电源金属外壳。
07安装 SATA 硬盘Install the SATA drive›
  1. 把硬盘放入对应尺寸的硬盘位。
  2. 使用螺丝或机箱免工具卡扣固定。
  3. 让接口朝向方便接线的一侧。
  4. 轻推硬盘，确认不会晃动。
机械硬盘含有精密机械部件，避免碰撞和跌落。
08连接主板与 CPU 供电Connect motherboard power›
  1. 把 24-pin ATX 插头接入主板长插座。
  2. 把 CPU 4-pin 或 8-pin 插头接入 CPU 附近的供电插座。
  3. 确认两个插头的锁扣都已扣住。
只接 24-pin 而漏接 CPU 供电，是“风扇可能转但无法正常启动”的常见原因。
09连接 SATA 数据与电源Connect SATA data and power›
  1. 用较窄的 SATA 数据线连接硬盘与主板 SATA 接口。
  2. 用较宽的 SATA 电源线连接硬盘与电源。
  3. 检查两种 L 形接口方向正确并插紧。
两条线，两种作用：数据线负责传输，电源线负责供电。
10连接前面板与风扇Connect front panel and fans›
  1. 根据主板标识连接 Power SW、Reset SW 和 LED。
  2. 连接前置 USB 和 HD AUDIO。
  3. 把机箱风扇接入 SYS_FAN 或 CHA_FAN。
  4. 整理线材，避免碰到风扇叶片。
不同主板的前面板针脚排列可能不同，应查看主板丝印或说明书，不能只靠线材颜色猜测。
11完成开机前检查Final inspection›
  1. 检查所有供电、数据线和风扇线。
  2. 确认内存和显卡已经完全插入。
  3. 确认机箱中没有散落螺丝和工具。
  4. 让另一名组员独立复查一次。
先检查，再通电。把故障排除在按下开机键之前。
12首次启动与 BIOS 检查First boot and BIOS check›
  1. 连接显示器、键盘和电源线。
  2. 确认显示器接在正确的视频输出接口。
  3. 开机，观察风扇、指示灯和屏幕信息。
  4. 进入 BIOS，检查 CPU、内存容量、硬盘和温度。
出现异味、烟雾、火花或异常声音时，立即切断电源并报告教师。

Troubleshooting

没有成功启动时，按症状检查

每次只改变一项，再重新测试，这样才能知道哪一项解决了问题。

完全不通电

检查插座、电源线、电源背部开关、24-pin 供电、CPU 供电和 Power SW 位置。

风扇转但没有画面

检查显示器输入源、视频线接口、内存是否插紧；有独立显卡时确认视频线位置。

找不到 SATA 硬盘

分别检查 SATA 数据线和 SATA 电源线，并在 BIOS 中查看硬盘是否被识别。

开机后很快关机

立即停止反复开机，检查散热器固定、硅脂、CPU_FAN 接线和 BIOS 温度。
