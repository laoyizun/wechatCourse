## 建议优先拍摄

如果暂时没有时间拍全套，优先准备以下 12 张：

| 文件名 | 中文拍摄内容 | 拍摄时注意 |
| --- | --- | --- |
| `pc-case-open-overview.jpg` | 打开机箱侧板后，对整个机箱内部拍一张全景照片。画面中尽量同时看到主板、电源、硬盘、内存和散热器。 | 横向拍摄，镜头与机箱侧面尽量保持平行，避免只拍到局部。 |
| `motherboard-overview.jpg` | 对主板正面拍一张完整照片，用于制作可点击的主板配件地图。 | 尽量从正上方垂直拍摄，不要让线材大面积遮挡主板。 |
| `cpu-top.jpg` | 拆下 CPU 后拍摄正面，清楚显示 Intel、Core i3 和具体型号等文字。 | 捏住 CPU 边缘，不要触摸底部触点；确保型号文字对焦清楚。 |
| `cpu-socket-open.jpg` | 打开 CPU 插槽的金属压盖，在没有安装 CPU 时拍摄整个插槽。 | 不要触碰插槽内部针脚，拍完及时盖好保护。 |
| `cpu-cooler.jpg` | 拍摄拆下来的 CPU 散热器，包括风扇、散热片和固定脚。 | 最好同时拍到四个固定脚；底部如果有旧硅脂，不要碰到其他物品。 |
| `ram-module.jpg` | 拍摄一条完整的 DDR3 内存，包括标签、金手指和中间缺口。 | 放在干净的非金属表面，正面横向拍摄，不要触摸金手指。 |
| `ram-slots.jpg` | 拍摄主板上的全部内存插槽，显示插槽颜色、卡扣和防呆凸点。 | 让插槽两端都进入画面；如果旁边有 DIMM 标识，也一起拍进去。 |
| `sata-drive.jpg` | 拍摄课堂机器实际使用的机械硬盘或 2.5 英寸 SATA SSD 整体。 | 正面拍摄，并尽量让硬盘外形和标签都清楚可见。 |
| `sata-drive-connectors.jpg` | 对硬盘尾部接口拍特写，同时拍到较短的 SATA 数据接口和较长的 SATA 电源接口。 | 接口要清晰，最好从能看出两个 L 形防呆缺口的角度拍摄。 |
| `motherboard-sata-ports.jpg` | 拍摄主板上的 SATA 接口区域，用于说明 SATA 数据线接在主板哪里。 | 接口和旁边的 SATA 编号或文字标识要同时进入画面。 |
| `power-supply.jpg` | 拍摄拆下来的电脑电源整体，包括外壳、风扇和伸出的线材。 | 横向拍摄，让学生能看出电源的大小、安装方向和主要线材。 |
| `front-panel-header.jpg` | 拍摄主板上连接开机键、重启键和指示灯的前面板针脚区域。 | 一定要拍清针脚旁边的 F_PANEL、PANEL 或类似文字；这是后续接线最需要放大的照片。 |
